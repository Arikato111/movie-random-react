# <p align="center">ดูอะไรดี ?</p>

นึกไม่ออกจะดูอะไรดี ต้องนี่เลย ดูให้เต็มอิมจุใจ ไม่ต้องคิด เพราะสุ่มเอา

## Get started

- clone repository
- `yarn` or `npm install`
- `yarn dev` or `npm run dev`

## Inside it

- React + Vite + Tailwindcss + TypeScript
- react-router-dom
- You can add movies at ./src/Data/Movies.json
